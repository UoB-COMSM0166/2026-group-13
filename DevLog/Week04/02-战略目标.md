<h2 id="strategy">🎯 战略目标</h2>

让玩家在  **录制 + 重现**  的规则下, 用 **简单** 的操作,体验 **脑洞大开** 的 **合作** 解谜巧妙感。

#### 关键词：
- 上手简单  
- 巧妙 / 脑洞
- 新鲜感  
- 游戏节奏（紧张 ↔ 放松、简单 ↔ 复杂）  
- 零挫败  
- 情绪反馈  
- 识别度 / 核心竞争力 / 不可替代性  
- 规则一致性

<h2 id="audience">📝 游戏受众分析</h2>

#### 1. 入口用户
- 不是因为机制，而是因为画风 **干净、可爱、有趣、新奇** 而进入游戏
- 轻度玩家，愿意尝试 10–30 分钟  
- 喜欢巧妙机制，但不追求深度

#### 2. 深度用户
- 喜欢巧妙、喜欢思考、喜欢规则突破  
- 与《割绳子》《切水果》《茶叶蛋大冒险》《uvsu》的受众群体 **重合度较高**
- 对“录制 + 重现”的机制天然敏感

<h2 id="data">📈 后台游戏数据</h2>

#### 第一阶段：入口表现
- 点击率高  
- 前 3 关通过率：**70%～80%**
- 前 3 关平均通关时长

#### 第二阶段：留存表现
- 第 1 天留存：**25%～35%**  
- 第 7 天留存：**5%～8%**

#### 第三阶段：深度参与
- 在线时长
- 自然分享率：**1%～2%**  
- 各个关卡通过率、通关时长


<h2 id="examples">💡 常见游戏的战略目标举例（参考部分）</h2>

#### 切水果

- 用“零门槛操作 + 即时爽感 + 高密度反馈”构建一个人人都能上手、人人都能爽的短时快感循环。

#### 合成大西瓜

- 用“极简规则 + 明确成长目标 + 随机性期待”构建一个轻度上瘾的合成循环，让玩家不断追求更大一合。

#### 枪弹辩驳

- 用“高张力叙事 + 角色魅力 + 推理参与感”构建一个情绪驱动的沉浸体验，让玩家在悬念与反转中持续投入。

#### 三角洲行动

- 用“真实战术体验 + 团队协作 + 高压对抗”构建一个长期可运营的战术竞技生态。

#### 第五人格

- 用“非对称对抗 + 强情绪体验 + 轻社交绑定”构建一个长期可运营的竞技生态，让玩家在紧张、恐惧、逃生、反杀中获得持续刺激。

#### 无畏契约

- 用“战术射击 + 英雄技能 + 高竞技上限”构建一个全球化的电竞生态，让玩家在高压对抗中追求极限操作与团队默契。

#### The Final Station

- 用“末日旅途 + 生存压力 + 片段式叙事”构建一个情绪沉浸体验，让玩家在不断前行中体验孤独、责任与未知。

#### 小婴儿逃出系列

- 用“反常识脑洞 + 轻恶搞 + 快速反馈”构建一个持续带来惊喜与笑点的轻解谜体验。
